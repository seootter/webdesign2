프로그램 실행 방법 학습

  • 웹 브라우저의 콘솔에서 실행
  • html 페이지에 삽입해 웹 브라우저로 실행
    • 코드를 html 태그 내 직접 추가하는 경우
    • 코드를 별도의 파일을 작성하고 파일을 html에 추가하는 경우

팁! 자바스크립트로 만든 프로그램은 일반적으로 불러오는데 시간이 오래 걸림

브라우저는 위에서 아래로 차례대로 내려가면서 태그를 해석하고, 태그에 맞게 동작을 함

브라우저가 특정 태그를 처리하는데 시간이 오래 걸린다면 그 밑에 작성한 태그들이 화면에 보이기 까지 시간이 오래 걸림
